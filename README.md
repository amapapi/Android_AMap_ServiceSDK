Android_AMap_ServiceSDK

高德搜索服务SDK介绍
<br /><h2> 简介：</h2>
<br /> Android搜索服务SDK 是实现了关键字搜索、周边查询、团购优惠查询、公交线路查询与规划、步行和驾车路径规划以及坐标\地址查询的功能。
